# ADR 006: Uso de GitHub como Repositorio Principal
Decisión técnica para control de versiones y colaboración.  
**Estado:** Aceptado  

## Contexto
El desarrollo del sistema requiere control de versiones, pull requests, issues  
y herramientas de integración automática.

## Decisión
Utilizar **GitHub** como repositorio central del proyecto.

## Alternativas Consideradas
- **GitLab:** Muy completo, pero se descartó por disponibilidad.  
- **Bitbucket:** Orientado más a empresas privadas.  

## Consecuencias
- Acceso fácil para los desarrolladores.  
- Gestión de ramas y PRs optimizada.  
- Integración con GitHub Actions.  
- Dependencia del ecosistema GitHub.
